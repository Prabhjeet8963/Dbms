# Entity-Relationship (ER) Modeling and Analysis

This directory contains conceptual database designs. It translates descriptive requirements into structured designs using Entity-Relationship diagrams.

To enable direct integration with developer platforms, all diagrams in this folder are modeled in Markdown using Mermaid.js, enabling them to render dynamically in your browser.

---

## Directory Index

1. [company-dbms/](./company-dbms/)
   Focuses on corporate associations, including recursive employee supervision hierarchies, many-to-many relationship attributes, and identifying relationships for weak entities.

2. [nhl-league/](./nhl-league/)
   Conceptual sports database tracking teams, players, captains, recursive host/guest game schedules, and weak injury records.

3. [university-registrar/](./university-registrar/)
   Models a university registrar system. Outlines weak entity hierarchies, circular prerequisite self-relations, and ternary grade report assignments.
